# 🎨 Fundamentos de CSS

Este repositório contém meus estudos iniciais de CSS, incluindo:

- Cores e fontes
- Propriedades de display
- Seletores
- Exercícios práticos

## 📁 Estrutura

- `cores-fontes/` → Estudo de cores e tipografia
- `display/` → Testes com display (block, inline, etc)
- `exercicio-css/` → Exercícios práticos
- `seletores/` → Tipos de seletores CSS

## 🚀 Objetivo

Praticar e consolidar fundamentos de CSS para desenvolvimento front-end.
